# 🚀 Application Tracker

A web-based platform designed specifically for students to manage and track their internship applications in one centralized location.

---

# 📖 Overview

Application Tracker helps students organize their internship search journey by providing a structured system to track applications, monitor deadlines, manage interview processes, and maintain application records.

Students often apply to internships through multiple platforms such as LinkedIn, Internshala, Unstop, company career portals, and campus opportunities. Managing these applications manually through spreadsheets or notes can become difficult and time-consuming.

This project aims to solve that problem by creating a dedicated platform where students can efficiently manage every stage of their internship application process.

---

# 🎯 Problem Statement

Students frequently face challenges such as:

- Missing application deadlines
- Forgetting interview schedules
- Losing track of application statuses
- Managing applications across multiple platforms
- Difficulty monitoring internship search progress

Application Tracker provides a centralized solution to organize and monitor all internship applications effectively.

---

# ✨ Key Features

## 📊 Dashboard

A centralized dashboard that provides an overview of:

- Total Applications
- Active Applications
- Upcoming Deadlines
- Recent Activities
- Internship Statistics

---

## ➕ Add Application

Students can store internship details including:

- Company Name
- Internship Role
- Application Date
- Deadline
- Current Status
- Additional Notes

---

## 📂 Application Tracking

Track applications through different stages:

- Applied
- Assessment
- Interview
- Selected
- Rejected

---

## 📝 Application Details

View complete information about each internship application:

- Company Information
- Application Timeline
- Current Status
- Notes
- Important Dates

---

## 👤 Student Profile

Manage personal information and maintain internship application history.
